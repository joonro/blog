Windows 10: How to Enable Dark mode in PowerShell

Joon Ro

2016-02-10 16:00

Comments

Source

With Windows 10 Anniversary Update, one can choose between the light and dark modes. Here are a way to set this in PowerShell in case you want to automate it. These are slightly modified commands from this reddit post.

To use the dark mode:

0
 
Set-ItemProperty -Path HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize -Name AppsUseLightTheme -Value 0

To use the light mode:

0
 
Set-ItemProperty -Path HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize -Name AppsUseLightTheme -Value 1
