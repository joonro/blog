Windows 관련된 부분 포함

1 hunspell 설치

가장 편한 방법은 Windows Package Manager인 Chocolatey 를 사용하는 것입니다: 

> choco install hunspell.portable

2 init

Emacs의 init file 에 다음을 추가합니다: 

(require 'ispell)

(cond ((eq system-type 'gnu/linux) 
    (progn
      (setq ispell-program-name "hunspell")
     ) 
  ((eq system-type 'windows-nt)
    (progn
       (add-to-list 'exec-path "C:/ProgramData/Chocolatey/bin/")
       (setq ispell-program-name (locate-file "hunspell"
                                              exec-path exec-suffixes 'file-executable-p))
     ))))

(setq ispell-really-hunspell t)

(add-to-list 'ispell-dictionary-alist '("en_US"
                                        "[[:alpha:]]"
                                        "[^[:alpha:]]"
                                        "[']"
                                        t
                                        ("-d" "en_US")
                                        nil
                                        utf-8))
(add-to-list 'ispell-dictionary-alist '("korean"
                                        "[가-힣]"
                                        "[^가-힣]"
                                        "[0-9a-zA-Z]"
                                        nil
                                        ("-d" "ko_KR")
                                        nil
                                        utf-8))

;; 한국어를 기본 사전으로 지정
(setq ispell-dictionary "korean")
; (setq ispell-dictionary "en_US")
